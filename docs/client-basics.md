## html5 (web markup language)

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>Title of the document</title>
</head>

<body>
  <div>
    <p>The content of the document...</p>
  </div>
</body>
</html>
```

```html
<div id="container">
  <header>
     <h1>Sample HTML5 Structure</h1>
     <nav>
       <ul>
         <li><a href="#">Home</a></li>
         <li><a href="#">About</a></li>
       </ul>
     </nav>
  </header>
  <section>
    <hgroup>
      <h1>Main Section</h1>
      <h2>This is a sample HTML5 Page</h2>
    </hgroup>
    <article>
      <p>This is the content for the first article</p>
    </article>
    <article>
      <p>This is the content for the second article</p>
    </article>
  </section>
  <footer>
    <p>This is the Footer</p>
  </footer>
</div>
```

## css3 (style sheet language)

```css
p {
  margin: 0 0 10px;
}

ul,
ol {
  padding: 0;
  margin: 0 0 10px 25px;
}

li {
  line-height: 20px;
}
```

```css
a {
  color: #0088cc;
  text-decoration: none;
}

a:hover,
a:focus {
  color: #005580;
  text-decoration: underline;
}
```

```css
.container {
  margin-right: auto;
  margin-left: auto;
  *zoom: 1;
}

#map_canvas img,
.google-maps img {
  max-width: none;
}
```

```css
.table thead th {
  vertical-align: bottom;
}

.table-hover tbody tr:hover > td,
.table-hover tbody tr:hover > th {
  background-color: #f5f5f5;
}
```
